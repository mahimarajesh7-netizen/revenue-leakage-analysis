# 🛒 Revenue Leakage Analysis — Olist Brazilian E-Commerce

## 📌 Project Overview
End-to-end analysis identifying and quantifying revenue leakage 
in a Brazilian e-commerce platform using real transaction data 
from 100,000+ orders across 2016–2018.

## 🛠️ Tech Stack
- PostgreSQL, SQL (CTEs, Window Functions)
- Python (Pandas, SQLAlchemy, Plotly)
- Jupyter Notebooks

## 🗺️ Project Phases
- [x] Phase 1 — Data Exploration & Quality Audit
- [ ] Phase 2 — SQL Revenue Leakage Analysis
- [ ] Phase 3 — Python Quantification
- [ ] Phase 4 — Dashboard & Visualizations
- [ ] Phase 5 — Business Recommendations

## ⚙️ Setup
1. Clone this repo
2. Copy `.env.example` to `.env` and fill in your DB credentials
3. Run `pip install -r requirements.txt`
4. Open notebooks in order starting from `notebooks/01_data_exploration.ipynb`
